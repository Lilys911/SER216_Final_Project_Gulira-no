# SER216 Final Project – Digital Parking Permit System

## Project Description
A digital parking permit system for a university environment.
Students and staff can register vehicles, request parking permits,
check available parking zones, and report parking issues.
Security staff can verify permits and update violation records.

## Workflow
Vehicle Registration → Permit Request → Zone Check → Permit Approval → Parking Verification

## Requirements Summary
**Functional:**
1. Register a vehicle using student/staff ID and vehicle details
2. Submit a parking permit request for a specific zone
3. Check real-time availability of parking zones
4. Security staff can verify permits and log violations

**Non-Functional:**
1. Permit status must load within 3 seconds
2. User data must be stored securely with role-based access
